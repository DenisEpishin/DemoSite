# DemoSite
Demo website of a hardware store

1. Create and activate virtual enviroment [optional]:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
2. Install dependencies:
    ```bash
    pip3 install flask
    pip3 install flask_sqlalchemy
    ```
3. Run:
    ```bash
    python3 fl3.py
    ```
